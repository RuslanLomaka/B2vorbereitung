3 Passiv

Grammatik-Guide: Das Passiv (B2 TELC Vorbereitung)

Übersicht

Hier findest du eine Übersicht über die wichtigsten Regeln zum Passiv. Wir nutzen einfache Erklärungen, damit du die Logik schnell verstehst.

1. Was ist das Passiv? (Aktiv vs. Passiv)

Im Aktiv ist die Person (das Subjekt) wichtig: Wer macht etwas? Im Passiv ist die Handlung oder das Objekt wichtig: Was passiert? Wer es macht, ist oft egal oder unbekannt.

  • Aktiv: Der Mechaniker repariert das Auto.
  • Passiv: Das Auto wird (vom Mechaniker) repariert.

2. Vorgangspassiv: Die Bildung

Das Vorgangspassiv beschreibt eine Aktion, die gerade passiert.

Formel: werden (konjugiert) + Partizip II (am Ende)

Zeitform Hilfsverb Beispiel
Präsens wird / werden Die Tür wird geschlossen.
Präteritum wurde / wurden Die Tür wurde geschlossen.
Perfekt ist ... worden Die Tür ist geschlossen worden.
Plusquamperfekt war ... worden Die Tür war geschlossen worden.
Futur I wird ... werden Die Tür wird geschlossen werden.

Wichtig: Im Perfekt Passiv nutzen wir immer worden, nicht geworden.

3. Passiv mit Modalverben

Wenn ein Modalverb dabei ist, rutscht das Passiv-Hilfsverb werden in den Infinitiv ans Satzende.

Formel: Modalverb + Partizip II + werden

  • Präsens: "Das Haus muss renoviert werden."
  • Präteritum: "Das Haus musste renoviert werden."
  • Perfekt: "Das Haus hat renoviert werden müssen."

4. Von oder Durch? (Der Täter im Passiv)

Wenn wir doch sagen wollen, wer die Aktion macht, nutzen wir Präpositionen:

  • von + Dativ: Für Personen oder Institutionen.
  • "Der Brief wurde von meinem Chef unterschrieben."
  • durch + Akkusativ: Für Mittel, Instrumente oder Ursachen.
  • "Die Stadt wurde durch ein Erdbeben zerstört."

5. Zustandspassiv (sein + Partizip II)

Das Zustandspassiv beschreibt das Ergebnis einer Handlung. Die Aktion ist fertig.

Formel: sein (konjugiert) + Partizip II

  • Vorgangspassiv: "Die Tür wird geschlossen." (Jemand drückt die Tür gerade zu).
  • Zustandspassiv: "Die Tür ist geschlossen." (Sie ist schon zu).

6. Passivfähigkeit und das "Man"

Nicht alle Verben können im Passiv stehen (z. B. haben, sein, wissen). Sätze mit man werden im B2-Test oft in Passiv umgeformt:

  • Aktiv: "Man baut hier eine neue Brücke."
  • Passiv: "Hier wird eine neue Brücke gebaut."

Zusammenfassung für die Prüfung

Typ Fokus Struktur
Vorgangspassiv Die Aktion werden + Partizip II
Zustandspassiv Das Ergebnis sein + Partizip II
Täter (Person) Wer? von + Dativ
Täter (Mittel) Womit? durch + Akkusativ
